# HTML5 Skelett

Dies ist ein einfaches Skelett für eine HTML5-WebApp ohne Firlefanz
Als Abhänigkeiten sind zur Zeit eingebunden

* Twitter Bootstrap (mit Theme)
* IntroJS (optional)
* Font-Awesome
* jQuery
* Underscore (optional)
* Backbone (optional)
* d3 (optional)
* Mousetrap (optional)
* Moment (mit Lokalisierung)
